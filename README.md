Expense Tracker Web Application

A full-stack expense management application built using Flask, SQLite, HTML, CSS and JavaScript.

Features:
- User Authentication (Login & Registration)
- Expense Management (Add, Edit, Delete Expenses)
- Custom Categories
- Budget Management
- Analytics Dashboard
- Multi-Currency Support
- CSV, Excel and PDF Export
- Expense Import from CSV
- Group Expense Splitting (Splitwise-like functionality)

Tech Stack:
- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript
- Pandas
- ReportLab
